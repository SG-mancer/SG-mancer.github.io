# just-the-docs-template
This website was build using the [just-the-docs template](https://github.com/new?template_name=just-the-docs-template&template_owner=just-the-docs) to create a [Jekyll] site.

For more information visit [just-the-docs github repo](https://github.com/just-the-docs/just-the-docs).

# local testing
I installed on my [windows subsystem for linux](https://ubuntu.com/desktop/wsl) [Jekyll] and [Bundler].

Then follow the following instructions:
1.  Change your working directory to the root directory of your site.
2.  Run `bundle install`.
3.  Run `bundle exec jekyll serve` to preview it at `127.0.0.1:4000`.
4.  Stop server with `ctrl-c`
